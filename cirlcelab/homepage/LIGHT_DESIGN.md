# Light Design Summary

## Overview

This document combines the light theme palette from `COLORS.md` with the design language extracted from the Claude / Anthropic references.

## Overall Tone

- Warm
- Editorial
- Calm
- Trustworthy
- Premium but restrained
- Knowledge-focused rather than flashy

## Style Keywords

- Warm paper background
- Soft minimalism
- Editorial typography
- Quiet premium
- Large solid color blocks
- Human, not sci-fi

## Visual Direction

### Background

- Use warm off-white and paper-like backgrounds
- White may be used as a supporting neutral for contrast, highlights, or inner surfaces
- Avoid relying on pure white as the default full-page background
- Avoid relying on strong gradients as the main surface language

### Typography

- Large serif headlines for authority and depth
- Clean sans-serif text for navigation, labels, and body copy
- Use size and spacing to build hierarchy before using color
- Let typography carry structure and confidence

### Layout

- Use generous whitespace
- Keep section rhythm clear and calm
- Favor wide, quiet compositions
- Make the page feel more like an editorial layout than a dashboard

### Color Use

- Use color as blocks, not glow
- Prefer low-saturation solid fills
- White may be used in small amounts to sharpen contrast and create breathing room
- Let neutrals dominate while muted colors group and emphasize content
- Use `#878785` and `#6A6B6A` as supporting neutrals for muted text, dividers, and quiet UI structure
- Good section or card colors:
  - `#bacfc8`
  - `#CAC9DA`
  - `#e9cccc`
  - `#e8e6dc`
  - `#cec6ba`

### Cards

- Prefer soft solid fills over heavy shadows
- Keep corner radius gentle, not playful
- Use borders only when structure needs them
- Color works better on inner content cards than on oversized outer wrappers

### Buttons

- Primary button: near-black fill with light text
- Secondary button: light fill with a subtle border
- Keep shapes simple, stable, and restrained
- Use a large rounded rectangle instead of a thin pill or sharp-cornered button
- Keep the silhouette calm and substantial, with generous horizontal padding
- Match the height and corner radius across primary and secondary actions
- Primary button details:
  - Fill: `#141413`
  - Text: `#ffffff` or `#faf9f5`
  - Border: none or visually invisible
  - Clay hover option: `#C96442`
  - Role: the strongest anchor on the page
- Secondary button details:
  - Fill: `#e8e6dc` or `#faf9f5`
  - Text: `#1F1E1D`
  - Border: `#cec6ba`
  - Role: calm companion action beside the primary button
- Avoid gradients, glows, glossy effects, or exaggerated shadows
- Hover states should only shift tone slightly, not change the button character

### Illustration Language

- Hand-drawn or slightly imperfect line work
- Symbolic, simple, memorable graphics
- Avoid glossy 3D effects, neon accents, or cyber styling

### Interaction

- Motion should stay subtle
- Hover feedback should feel crisp, not loud
- Avoid glow, high saturation, and dramatic movement

## Design Principles

1. Clarity over spectacle
2. Typography should carry authority
3. Color should separate sections and layers, not color everything at once
4. Use black sparingly but decisively for anchors such as buttons and key actions
5. Keep the interface human, thoughtful, and non-template-like

## One-Line Summary

A warm paper-like background, editorial black typography, restrained interface structure, and low-saturation solid color blocks used with confidence.
